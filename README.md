## React Native UI Challenge

oject Overview
This project is a high-fidelity front-end implementation of a mobile UI design using React Native.
The goal was to translate a static design into a functional, responsive, and maintainable mobile interface, focusing on layout accuracy, typography, and reusable component architecture.

Features
-
Pixel-Perfect Implementation: Close adherence to the provided design's layout, color palette, and spacing.

Reusable Components: Modular architecture for buttons, input fields, and cards to ensure DRY (Don't Repeat Yourself) code.

Responsive Design: Implemented using Flexbox and Dimensions API to ensure the UI looks great on various screen sizes (iOS/Android).

Interactive Elements: Touchables and buttons with active opacity states for better UX.

Tech Stack & Tools
-
Framework: React Native (CLI or Expo)

Styling: React Native StyleSheet (Flexbox)

Icons: [e.g., Lucide-react, FontAwesome, or Material Icons]

Fonts: [e.g., Roboto, Inter, or System Fonts]

Installation & Setup
-

1. git clone https://github.com/Ndima-karabo17/React-Native-UI-Challenge.git
2. npm install  to install dependencies
3. npm install react
4. npx expo start to run the application
5. Scan the QR Code using your expo go app in your android
